# Poke Api

En este proyecto hacemos peticiones HTPP a la api de PokéAPI, con la libreria Axios

### Notas:

Recueden reconstruir los módulos de Node

```
yarn install

```

Y para construir el build, recuerden:

```
yarn build
```
